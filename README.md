# gamify
Inkscape extension
## EN - 🧩 Gamify — Inkscape Extension for Creating Games

Gamify is an Inkscape extension that transforms your drawings into games.

<img width="533" height="200" alt="menuDetail" src="https://github.com/user-attachments/assets/e919b407-b3d9-46fd-b6e9-8ab3e9937e2c" />

With this extension, you can use:

**Color this drawing** — to create coloring games

**Highlight sequence** — to highlight elements in a specific order

**Find all the…** — to create “find the objects” games inside an image

**Match item** — to build matching or correspondence games

**Interactive story** — to create interactive stories

**Custom** — to generate your own game logic

### 🎯 Purpose of Gamify
Gamify aims to let any Inkscape user create games without writing code, except for the **Interactive story** and **Custom** options, which allow advanced users to add their own logic.

### 🧩 Writing Code in Gamify

If you want to customize your games further, the **Interactive story** and **Custom** options let you add your own lines of code using the FuncSug language.

This is optional: all other options work without any coding.
